sm_profile¶

Manage ShakeMap profiles.

These ShakeMap profiles allow one user to have multiple ShakeMap profiles (installations) on the same machine. These profiles can all be configured differently, if desired.

usage: sm_profile [-h] [-s PROFILE] [-l] [-c PROFILE] [-d PROFILE] [-a] [-p]

Named Arguments¶

-s, --switch Switch from current profile to PROFILE.
-l, --list

List all configured profiles.

Default: False

-c, --create Create new profile PROFILE and switch to it.
-d, --delete Delete existing profile PROFILE.
-a, --accept

Accept the defaults when creating or deleting a profile.

Default: False

-p, --preserve

Preserve data when deleting a profile

Default: False
