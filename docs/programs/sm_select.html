sm_select ++========

Parses the output of STREC in accordance with the configuration file, creates a new GMPE set specific to the event, and writes model_zc.conf in the event’s ‘current’ directory.

Configuration file: select.conf

usage: sm_select [-h] [-v] eventid

Positional Arguments¶

eventid ID of the event to process

Named Arguments¶

-v, --verbose Print informational messages.
